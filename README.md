## 플레이 방법
**이 게임은 아무것도 하지 않습니다!**
단지 켜놓은 상태에서 키보드나 마우스 입력이 있으면 고양이가 **돌아갑니다!**

- 고양이 왼쪽 상단에 마우스를 올리면 빨간 버튼과 노란 버튼이 있습니다.
- 빨간 버튼을 누르면 게임이 종료됩니다.
- 노란 버튼을 누르면 설정창이 열립니다.
    - Transparent 옵션을 끄면 초록 배경이 생깁니다.
    - Always On? 옵션을 끄면 고양이가 다른 창 뒤로 숨을 수 있습니다.

## 지원 환경
- Windows 10/11 (x64)
- (주의) macOS / Linux(프로톤 포함) 환경은 지원하지 않습니다.

## 입력 감지 방식
이 앱은 고양이를 돌리기 위해 **“입력이 발생했다”는 신호만** 감지합니다.

- 기본 동작: 앱(게임) 창이 활성화되어 있을 때 입력을 감지합니다.
- (옵션) **백그라운드 입력 감지**를 켜면, 다른 창을 사용 중이어도 입력 발생 시 고양이가 돌아갑니다.
  - 백그라운드 입력 감지는 OS 수준의 입력 이벤트를 사용합니다.
  - **마우스 이동은 감지하지 않도록(클릭/키 입력만) 처리**되어 있습니다. *(설정에 따라 달라질 수 있음)*

## 개인정보 / 보안 안내
- 이 앱은 **어떤 키를 눌렀는지(키 값/문자), 어떤 내용을 입력했는지**를 수집하지 않습니다.
- 입력은 **저장하지 않으며**, 외부로 **전송하지 않습니다**.
- 오직 “입력이 있었음”만 카운트/트리거로 사용하여 고양이 애니메이션을 재생합니다.

> 참고: 일부 보안 프로그램(백신/회사 PC 정책)에서 “백그라운드 입력 감지” 기능을 민감하게 볼 수 있습니다.  
> 이런 경우 설정에서 해당 옵션을 끄면 정상 동작합니다.
