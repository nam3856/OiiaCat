## OIIA OIIA

### 플레이 방법
**이 게임은 아무것도 하지 않습니다!**
단지 켜놓은 상태에서 키보드나 마우스 입력이 있으면 고양이가 **돌아갑니다!**

- 고양이 왼쪽 상단에 마우스를 올리면 빨간 버튼과 노란 버튼이 있습니다.
- 빨간 버튼을 누르면 게임이 종료됩니다.
- 노란 버튼을 누르면 설정창이 열립니다.
    - Transparent 옵션을 끄면 초록 배경이 생깁니다.
    - Always On? 옵션을 끄면 고양이가 다른 창 뒤로 숨을 수 있습니다.
